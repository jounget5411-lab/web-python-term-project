# 🎯 실전 모의투자 시스템 - 사용 가이드

## 빠른 시작

```bash
cd mock-investing
poetry install          # 최초 1회만
poetry run start        # 프로그램 실행
```

## 📖 단계별 가이드

### Step 1: 계좌 설정 (최초 1회)

```
메인 메뉴에서 1 선택 → 계좌 관리
  → 2 선택 → 입금
  → 원하는 금액 입력 (예: 1000000)
  → 0 선택 → 메인 메뉴로
```

**추천 초기 자금:**
- 연습용: 100만원
- 실전 시뮬레이션: 1000만원
- 고액 테스트: 1억원

---

### Step 2: 종목 선택

프로그램에서 제공하는 10가지 인기 종목 또는 직접 티커 입력

#### 한국 주식 (추천)
- 1: 삼성전자 - 한국 대표 반도체 기업
- 2: 카카오 - IT/플랫폼
- 3: 네이버 - IT/플랫폼
- 4: SK하이닉스 - 반도체
- 5: LG화학 - 화학/배터리

#### 미국 주식
- 6: 애플 (AAPL) - 아이폰
- 7: 테슬라 (TSLA) - 전기차
- 8: 구글 (GOOGL) - 검색/AI
- 9: 마이크로소프트 (MSFT) - 소프트웨어
- 10: 엔비디아 (NVDA) - GPU/AI

#### 직접 입력
- 한국: 종목코드.KS (예: 005930.KS)
- 미국: 티커 (예: AAPL, TSLA)

---

### Step 3: 전략 선택

#### 초보자 추천
**1. SMA 크로스오버 (5/20)**
- 가장 간단하고 직관적
- 단기(5일) > 장기(20일) → 매수
- 안정적인 수익 추구

#### 중급자 추천
**3. RSI 전략**
- RSI < 30: 과매도 → 매수
- RSI > 70: 과매수 → 매도
- 역추세 매매

**4. MACD 전략**
- MACD선 > 시그널선 → 매수
- 추세 추종

#### 고급자
**5. 볼린저 밴드**
- 변동성 활용
- 하단밴드 돌파 → 매수

**6. 모멘텀**
- 강한 추세 추종
- 고위험 고수익

---

### Step 4: 매매 설정

#### 수수료율
- **기본: 0.05% (0.0005)**
- 한국 증권사 평균: 0.015%
- 미국 주식: 0%

#### 쿨다운 시간
- **기본: 1일**
- 의미: 마지막 거래 후 N일 후에 다시 거래
- 과도한 매매 방지

#### 1회 주문 비율
- **기본: 20% (0.2)**
- 의미: 현재 잔액의 20%씩 매수
- 권장: 10-30%

---

### Step 5: 결과 분석

#### 수익률 평가
- **-10% 이하**: 전략 재검토 필요
- **-10% ~ 0%**: 손실이지만 경험 축적
- **0% ~ 10%**: 양호
- **10% ~ 20%**: 우수
- **20% 이상**: 매우 우수 (또는 운이 좋음)

#### 거래 횟수 평가
- **1-5회**: 너무 적음, 전략이 신호를 잘 안냄
- **5-20회**: 적절
- **20-50회**: 많음, 수수료 확인
- **50회 이상**: 과도한 매매, 쿨다운 증가 필요

## 💡 실전 팁

### Tip 1: 종목 선택
- **상승장**: 삼성전자, 테슬라 같은 성장주
- **하락장**: 안정적인 대형주
- **횡보장**: RSI, 볼린저 밴드 전략 유리

### Tip 2: 기간 선택
- **1개월**: 단기 변동성 확인
- **3개월**: 중기 트렌드 파악 (추천)
- **6개월**: 장기 안정성 검증
- **1년**: 전체 사이클 확인

### Tip 3: 전략 조합
1. 같은 종목, 다른 전략으로 여러 번 테스트
2. 가장 높은 수익률 전략 선택
3. 여러 종목에 같은 전략 적용해보기

### Tip 4: 리스크 관리
- 1회 주문 비율 낮게 (10-20%)
- 쿨다운 시간 넉넉히 (1-3일)
- 손실 -10% 이상 시 전략 변경

## 📊 사용 예시

### 예시 1: 삼성전자 SMA 전략 (3개월)

```
1. 계좌 관리 → 100만원 입금
2. 모의투자 시작
3. 종목: 1 (삼성전자)
4. 기간: 2 (3개월)
5. 전략: 1 (SMA)
6. 설정: 모두 Enter (기본값)
7. 결과 확인 → 차트 보기
```

**예상 결과:**
- 거래: 8-12회
- 수익률: 5-15% (시장 상황에 따라)

---

### 예시 2: 테슬라 MACD 전략 (6개월)

```
1. 모의투자 시작
2. 종목: 7 (테슬라)
3. 기간: 3 (6개월)
4. 전략: 4 (MACD)
5. 설정:
   - 수수료: 0 (미국은 수수료 무료)
   - 쿨다운: 2 (2일)
   - 주문 비율: 0.15 (15%)
6. 결과 확인
```

**예상 결과:**
- 거래: 15-25회
- 수익률: -10% ~ +30% (변동성 큼)

---

### 예시 3: 카카오 RSI 전략 (1년)

```
1. 모의투자 시작
2. 종목: 2 (카카오)
3. 기간: 4 (1년)
4. 전략: 3 (RSI)
5. 결과 → 캔들스틱 차트
```

**예상 결과:**
- 거래: 20-30회
- 수익률: 0-20%
- RSI는 횡보장에 강함

## ⚠️ 주의사항

### 반드시 기억하세요!
1. **과거 성과 ≠ 미래 수익**
   - 백테스팅은 과거 데이터 기반
   - 실제 투자 시 결과는 다를 수 있음

2. **데이터 지연**
   - 무료 API는 15-20분 지연
   - 실시간 투자와 차이 있음

3. **수수료 영향**
   - 거래 많으면 수수료 누적
   - 실제 거래 시 더 중요

4. **감정 제어**
   - 모의투자는 감정 없음
   - 실제 돈이면 감정 개입

5. **과적합 위험**
   - 특정 기간에만 잘 작동할 수 있음
   - 여러 기간, 종목으로 검증 필요

## 🆘 문제 해결

### Q1. 프로그램이 안 열려요
```bash
poetry --version  # Poetry 설치 확인
poetry install    # 의존성 재설치
poetry run start  # 실행
```

### Q2. 종목 데이터를 못 가져와요
- 인터넷 연결 확인
- 티커 오타 확인
- 나중에 다시 시도 (API 제한)

### Q3. 차트가 안 보여요
- matplotlib 설치 확인
- 한글 폰트 없어도 차트는 보임

### Q4. 수익률이 너무 낮아요
- 다른 전략 시도
- 다른 기간 시도
- 다른 종목 시도
- 매개변수 조정

### Q5. 거래가 너무 많아요/적어요
- 쿨다운 시간 조정
- 다른 전략 선택
- 기간 변경
