# 📝 변경 내역

## v2.2 (2025-11-22) - 긴급 수정

### 🔧 주요 버그 수정

#### 1. 수수료율 입력 방식 수정
**문제:** 사용자가 0.05 입력 시 5%로 인식되는 심각한 오류
```
Before: 0.05 입력 → 5% (100배 차이!)
After:  0.05 입력 → 0.05% (정상)
```

**해결:**
- % 단위로 입력받도록 명확히 표시
- 입력값을 100으로 나누어 변환
- 예시 추가: "0.05 입력 = 0.05%"

#### 2. 거래 발생 안 하는 문제 진단 강화
**문제:** 거래가 안 되는데 이유를 알 수 없음

**해결:**
- 실시간 신호 추적 시스템 추가
  ```
  • BUY 신호 발생: 15회
  • SELL 신호 발생: 8회
  • 쿨다운에 막힘: 12회
  • 보유 자산 없음: 8회
  • 현금 부족: 0회
  ```
- 주요 원인 자동 분석
- 맞춤형 해결책 제시

#### 3. 전략 설명 대폭 강화
**Before:**
```
1. SMA 크로스오버 (5/20) - 초보자 추천
```

**After:**
```
1. SMA 크로스오버 (5/20) - 초보자 추천 ⭐
   📌 단기(5일) 평균이 장기(20일) 평균을 뚫으면 신호
   ✅ 매수: 5일 평균 > 20일 평균 (상승 추세)
   ❌ 매도: 5일 평균 < 20일 평균 (하락 추세)
   💡 적합: 추세가 명확한 종목
```

- 언제 매수/매도하는지 명확히 표시
- 어떤 종목에 적합한지 안내
- 6가지 전략 모두 상세 설명 추가

#### 4. 기본값 조정 (더 공격적으로)
**Before:**
- 쿨다운: 1일 (보수적)
- 주문 비율: 0.2 (20%)

**After:**
- 쿨다운: 0일 (신호마다 거래) ⚡
- 주문 비율: 0.3 (30%)

**이유:** 백테스팅에서는 더 많은 거래가 발생해야 전략 효과를 제대로 볼 수 있음

#### 5. 디버깅 로그 추가
```
🔄 백테스팅 실행 중...
   데이터: 242일
   전략: SMA 크로스오버 (5/20)
   쿨다운: 0일
```

- 실행 전 설정 확인 가능
- 첫 거래 체결 시 즉시 알림
- 5건마다 진행상황 표시
